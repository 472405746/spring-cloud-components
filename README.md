# spring-cloud-componets

一款基于 SpringCloud 组件构建的分布式服务架构（待完善），可以作学习之用

## 系统module

### eureka-service

SpringCloud 的服务注册中心

### spring-demo-service

一个服务实例demo

### spring-demo-service-ribbon

基于 rest + ribbon 实现的拥有负载均衡的服务消费者

### spring-demo-service-feign

基于 Feign 实现的服务消费者， Feign 是基于 ribbon 实现的， 已实现负载均衡
